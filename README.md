# Fake News Detection Using BERT

## Project Overview
This project uses the BERT (Bidirectional Encoder Representations from Transformers) model to classify news headlines as Real or Fake.

## Technologies Used
- Python
- Google Colab
- PyTorch
- Transformers
- Scikit-Learn

## Features
- Fake News Classification
- BERT-based NLP Processing
- Real-Time Prediction
- Performance Evaluation

## Results
- Accuracy: 98.87%
- Precision: 99%
- Recall: 99%
- F1-Score: 99%
